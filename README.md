# 🤖 Jarvis AI Assistant

An intelligent AI-powered voice assistant built with **Python**, **Groq Llama 3.3 70B**, and a modular skill-based architecture. Jarvis combines conversational AI with desktop automation, enabling users to interact naturally through voice while performing real-world tasks.

---

## 🚀 Features

- 🎙️ Voice-based interaction
- 🧠 Powered by Groq Llama 3.3 70B
- 💬 Natural language conversations
- 🔊 Text-to-Speech responses
- 🎤 Speech Recognition
- ⚡ Wake Word Detection
- 🧩 Modular Skill System
- 🌐 Open websites using voice
- 💻 Launch desktop applications
- 📷 Camera access
- 📸 Screenshot capture
- 🖥️ Windows automation
- 📝 Context-aware conversations
- 🔌 Easily extensible architecture

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/ANNEMHEMANTHKUMAR/Jarvis.git
```

### Navigate to the project

```bash
cd Jarvis
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Configure API Key

Create a `.env` file and add:

```env
GROQ_API_KEY=YOUR_API_KEY
```

---

## ▶️ Run the Project

```bash
python main.py
```
